CSS Animation Examples

This page demonstrates various CSS animation and transition techniques. Hover over or click on elements to see the animations in action.

Example 1: Simple Button Transition

This button has transitions on its background color and size. Hover over the button to see the transition effect.

/* CSS */
.button-1 {
    background-color: #3498db;
    color: white;
    padding: 12px 24px;
    border-radius: 4px;
    /* Transition */
    transition: background-color 0.3s ease, transform 0.2s ease;
}

.button-1:hover {
    background-color: #2980b9;
    transform: scale(1.05);
}

Example 2: Color Change and Shape Transition

This box changes color and shape when hovered. Hover over the square to see it transform into a circle with a different color.

/* CSS */
.color-box {
    width: 100px;
    height: 100px;
    background-color: #e74c3c;
    /* Transition */
    transition: background-color 0.5s ease, border-radius 0.5s ease-in-out;
}

.color-box:hover {
    background-color: #9b59b6;
    border-radius: 50%;
}

Example 3: Pulse Animation using Keyframes

This box continuously pulses using CSS keyframe animation. The animation runs infinitely and doesn't require any user interaction.

/* CSS */
@keyframes pulse {
    0% {
        transform: scale(1);
    }
    50% {
        transform: scale(1.2);
    }
    100% {
        transform: scale(1);
    }
}

.pulse-box {
    width: 100px;
    height: 100px;
    background-color: #2ecc71;
    animation: pulse 1.5s ease-in-out infinite;
}

Example 4: Loading Spinner

A common loading spinner animation created using a continuous rotation animation.

/* CSS */
@keyframes spin {
    0% { 
        transform: rotate(0deg); 
    }
    100% { 
        transform: rotate(360deg); 
    }
}

.spinner {
    width: 60px;
    height: 60px;
    border: 6px solid #f3f3f3;
    border-top: 6px solid #3498db;
    border-radius: 50%;
    animation: spin 1s linear infinite;
}

Example 5: Slide-in Animation

This box slides in from the left side of the screen using a keyframe animation.

/* CSS */
@keyframes slideIn {
    from {
        transform: translateX(-100%);
        opacity: 0;
    }
    to {
        transform: translateX(0);
        opacity: 1;
    }
}

.slide-box {
    width: 100px;
    height: 100px;
    background-color: #f39c12;
    animation: slideIn 1s ease-out forwards;
}

Example 6: Bounce Animation

A multi-step bounce animation using multiple keyframe points.

/* CSS */
@keyframes bounce {
    0%, 20%, 50%, 80%, 100% {
        transform: translateY(0);
    }
    40% {
        transform: translateY(-30px);
    }
    60% {
        transform: translateY(-15px);
    }
}

.bounce-box {
    width: 60px;
    height: 60px;
    background-color: #e67e22;
    border-radius: 50%;
    animation: bounce 2s infinite;
}

Click on the card to see a 3D flip animation using backface-visibility and transform-style properties.

This is an example of 3D transforms in CSS. 
/* CSS */
.card-container {
    width: 200px;
    height: 300px;
    perspective: 1000px; /* Creates 3D effect */
}

.card {
    width: 100%;
    height: 100%;
    position: relative;
    transition: transform 0.8s;
    transform-style: preserve-3d;
}

.card.flipped {
    transform: rotateY(180deg);
}

.card-front, .card-back {
    position: absolute;
    width: 100%;
    height: 100%;
    backface-visibility: hidden;
}

.card-back {
    transform: rotateY(180deg);
}
